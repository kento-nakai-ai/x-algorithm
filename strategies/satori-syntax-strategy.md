# さとり構文活用戦略 - 万バズ量産のための文章構成法

## 🎯 概要

**名称**: さとり構文  
**目的**: SNSで共感を呼び自然と拡散される文章構成  
**効果**: わずか140文字で万バズを量産、ターゲット層のフォロワー爆増  
**原理**: 潜在的ニーズを刺激し、行動を誘発するマーケティング手法

## 🧠 さとり構文の基本原理

### 心理学的根拠
1. **平均以上バイアス**: 誰しも「自分は平均以上」と思う認知の歪みを活用
2. **コールドリーディング**: 曖昧な表現で多くの人に当てはまると感じさせる
3. **認知的不協和の解消**: 矛盾する認知から生じる不快感を解消する心理
4. **プロスペクト理論**: 損失回避性を活用したブックマーク誘発

### 拡散メカニズム
```
フォロワー → ターゲット層 → マス層
    ↓           ↓          ↓
エンゲージ → 有益判定 → おすすめ掲載
```

## 📝 さとり構文の基本構造

### さとり構文A（基本形）
```
[属性・能力]の人は全員[道具・ツール]を[動詞]。
[ベネフィット][具体的な数字]。
[冒頭と逆の属性・能力]も[理想の未来]。
```

### 構成要素詳細

#### 1. 属性・能力
- **例**: 「超深い思考ができる人」「文章得意な人」
- **ポイント**: 平均以上バイアスで広範囲にターゲティング
- **注意**: 主観的指標で多くの人が「自分のこと」と思える表現

#### 2. 道具・ツール・作業
- **例**: 「AIツール」「動画編集」
- **ポイント**: ニッチなツールでも問題なし
- **効果**: 「知らないこと」として有益性を感じさせる

#### 3. 左に繋がる動詞
- **基本**: 「使った方が良い」
- **逆パターン**: 「使わない方が良い」（さとり構文B）
- **ニュース系**: 「朗報」（さとり構文C）

#### 4. ベネフィット
- **重要度**: ★★★★★
- **ポイント**: ターゲットのインサイト（購買意欲の核心）に基づく
- **注意**: 自分の主観ではなく徹底的にターゲット視点

#### 5. 具体的な数字
- **例**: 「10倍」「100万円」
- **注意**: あってもなくても良い、適当でOK

#### 6. 冒頭と逆の属性・能力
- **目的**: カバー率100%を目指す
- **効果**: より多くの人に刺さる投稿になる

#### 7. 理想の未来
- **重要度**: ★★★★☆
- **ポイント**: 抽象的ではなくインサイトに基づく具体的表現
- **効果**: 行動を起こすモチベーションを提供

## 🛠 さとり構文のバリエーション

### さとり構文B（否定形）
```
[ターゲット]は[ツール名]を使わない方が良い。
[理由・エビデンス]
[代替案とベネフィット]
```

**特徴:**
- 「〜を使わない方が良い」で専門性をアピール
- 認知的不協和とプロスペクト理論を活用
- エンゲージメント率が高い傾向

### さとり構文C（ニュース形）
```
[ターゲット]に朗報。
[ニュース内容とベネフィット]
[補足情報]
```

**特徴:**
- ニュース性の高い内容に適用
- ブックマークを誘発しやすい
- 速報性がエンゲージメントを高める

### さとり構文D（口語調）
```
[口語調の興奮表現]
[ニュース内容]
[詳細情報]
```

**特徴:**
- 「隣の人にスマホ画面を見せるように」
- 臨場感と興奮が伝わる表現
- 前半口語調、後半ニュース調のハイブリッド

### さとり構文E（ステップ形）
```
[ターゲット設定]
・[ステップ1]
・[ステップ2]
・[ステップ3]
[CTA（行動喚起）]
```

**特徴:**
- 箇条書きでステップを明示
- 副業系やハウツー系に最適
- CTAで行動を促進
