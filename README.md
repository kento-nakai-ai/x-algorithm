## 🎯 概要

𝕏（旧Twitter）での効果的な集客戦略とアルゴリズムの理解を深めるための包括的なWebプレゼンテーション集です。AI界隈のマスコット「わど」の運用戦略から最新アルゴリズムの解説まで、実践的な内容をインタラクティブなスライド形式で提供します。

## 🎮 使用方法

### 基本操作
- **次のスライド**: `→` キー または `スペース` キー または「次へ」ボタン
- **前のスライド**: `←` キー または「前へ」ボタン
- **スクロール**: マウスホイール または スライド内のスクロールバー

### AI機能の使用（アルゴリズム解説のみ）
1. スライド8の「✨ アイデアを生成」ボタンをクリック
2. 対応する運用戦略に基づいたツイート案が自動生成
3. 生成されたツイートはTwitterライクなプレビューで表示

> **注意**: AI機能を使用するにはGemini APIキーが必要です。

## 🏠 ローカル開発

### 前提条件
- モダンなWebブラウザ（Chrome, Firefox, Safari, Edge）
- ローカルサーバー（開発時推奨）

### セットアップ
```bash
# リポジトリをクローン
git clone https://github.com/kento-nakai-ai/x-algorithm.git
cd x-algorithm

# ローカルサーバーを起動（例：Python）
python -m http.server 8000
# または Node.js の場合
npx serve .

# ブラウザで http://localhost:8000 を開く
```

### Gemini AI機能の設定
```javascript
// algorithm/index.html の該当箇所を編集
const apiKey = "YOUR_GEMINI_API_KEY_HERE";
```

## 📊 𝕏アルゴリズムの要点

### ✅ 高評価される要素
- **リプライへのエンゲージ** - 最大ブースト効果
- **リプライ（返信）** - 会話の促進
- **長時間閲覧** - 2分以上の滞在時間
- **プロフィールクリック** - ユーザーの関心度

### ❌ 低評価される要素
- **通報・ブロック** - 絶対回避
- **外部リンククリック** - 𝕏からの離脱
- **ネガティブフィードバック** - 「興味なし」報告
- **ハッシュタグ乱用** - 3つ以上は要注意

## 📈 集客戦略の核心

### 🎯 Daily Routine（共通基盤）
- **朝**: AIニュース速報投稿
- **昼**: 戦略的リプライ（5件）
- **夜**: メインコンテンツ投稿
- **随時**: 全リプライへの返信＆集客

### 📊 KPI管理
- **最重要指標**: LINEリスト増加数
- **管理指標**: インプレッション、エンゲージメント率、プロフィールクリック数

### 🚀 成長フェーズ
1. **短期スプリント**（7月）: 高密度アクションで急成長
2. **持続的成長**（8月）: 計画的テーマ投稿で安定成長
