## 🎯 概要

𝕏（旧Twitter）での効果的な集客戦略とアルゴリズムの理解を深めるための包括的なWebツール・プレゼンテーション集です。AI界隈のマスコット「わど」の運用戦略、なかいけんとの専用インタラクティブツール、最新アルゴリズムの解説、Discordチャンネル分析による実証済みバズパターン、万バズを量産する「さとり構文」戦略、チーム運用の議事録まで、実践的な内容をスライド形式とWebアプリケーション形式で提供します。

## 🎮 使用方法

### 基本操作
- **次のスライド**: `→` キー または `スペース` キー または「次へ」ボタン
- **前のスライド**: `←` キー または「前へ」ボタン
- **スクロール**: マウスホイール または スライド内のスクロールバー

### AI機能の使用（アルゴリズム解説のみ）
1. スライド8の「✨ アイデアを生成」ボタンをクリック
2. 対応する運用戦略に基づいたツイート案が自動生成
3. 生成されたツイートはTwitterライクなプレビューで表示

> **注意**: AI機能を使用するにはGemini APIキーが必要です。

## 🏠 ローカル開発

### 前提条件
- モダンなWebブラウザ（Chrome, Firefox, Safari, Edge）
- ローカルサーバー（開発時推奨）

### セットアップ
```bash
# リポジトリをクローン

# ローカルサーバーを起動（例：Python）

```

## 📊 𝕏アルゴリズムの要点

### ✅ 高評価される要素
- **リプライへのエンゲージ** - 最大ブースト効果
- **リプライ（返信）** - 会話の促進
- **長時間閲覧** - 2分以上の滞在時間
- **プロフィールクリック** - ユーザーの関心度

### ❌ 低評価される要素
- **通報・ブロック** - 絶対回避
- **外部リンククリック** - 𝕏からの離脱
- **ネガティブフィードバック** - 「興味なし」報告
- **ハッシュタグ乱用** - 3つ以上は要注意
